== Description ==
An elegant WordPress slider, simple and ready to use.

<h4>Features</h4>
* Create Unlimited Sliders with unlimited slides.
* Insert Sliders anywhere on your site using a simple shortcode.
* WP Slider is configurable.
	
== Installation == 
* In your Admin, go to menu Plugins > Add
* Search for bSlider
* Click to install
* Activate the plugin
